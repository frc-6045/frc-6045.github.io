Documented Problems page

We have experienced these problems with our robot and struggled to fix them, so we are writing them here for future reference and in case they are useful for other teams.

Some swerve modules occasionally rotate uncontrollably or are affected by other modules' encoder values

      Ensure that the encoders of the affected modules are plugged in.


The Limelight can be accessed through the online dashboard, but not through the code

      Ensure that the Custom NT Server IP field in the Settings tab of the online dashboards has the same IP address as the RoboRIO.
